## Quadropedal Robot Description

### Prerequisites
- ROS 1
- RVIZ
- Robot State Publisher
- Joint State Publisher GUI

### Installation 

```bash
$ cd ~/catkin_ws/src
$ git clone https://github.com/mjlee111/lux_description.git
$ cd ..
$ catkin_make
```

### Launch
```bash
$ roslaunch lux_description lux.launch
```
